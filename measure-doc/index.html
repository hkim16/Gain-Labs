Technical documentation for the measures that compound the Gain Index.

Description

This site documents the measure choices for the Gain Index. The GAIN Index captures a country's Vulnerability to climate change and other global challenges, and its Readiness to improve resilience. Each of these 2 axis is composed of several measures.

Rationale

Measures were selected according to the following criteria:

  • Geographic coverage.
  • Transparent and open.
  • Structured and modular.
  • Actionable.
  • Temporal.
  • Scalable.

Sources
Each measure candidate was drafted according to the criteria above, and then vetted thorugh an internal and external consultation process that included experts from academia, public and private sector.
